# Basic Master Data Module (BAS) Frontend

이 디렉토리는 전사 공통 기준 정보(BAS) 모듈의 프론트엔드 화면과 리소스를 포함합니다.
MM(자재), SD(영업), FI(재무), HR(인사) 등 모든 하위 모듈에서 공통으로 참조하는 **조직(회사/사업장/부서), 거래처(Business Partner), 환율 등 핵심 마스터 데이터** 관리를 전담합니다.

## 📌 화면 네이밍 룰 (Naming Convention)

- **디렉토리:** 대분류 업무 코드 (예: `BAST01` ~ `BAST05`)
- **파일 네이밍:** 대분류 업무 코드 + `V` (View) + 일련번호 3자리 (예: `BAST01V001.vue`)
  - `V001`: 주로 목록 조회 (List)
  - `V002`: 주로 상세 조회 및 등록/수정 (Detail/Form)
  - `V003~`: 특수 목적 화면 (트리 뷰, 대량 업로드 등)
