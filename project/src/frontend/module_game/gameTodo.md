## 🚧 진행 중 및 예정된 작업 (In Progress / Todo)

### Phase 1: 아키텍처 고도화 및 무한한 확장성 확보 (Architecture & Scalability)
- [x] **의존성 주입(DI) 폴더 구조 구축:** 모든 파일에 의존성을 완전히 독립적으로 주입할 수 있도록 폴더 경로 구축 완료. (`di/container`, `di/providers`, `di/interfaces`, `di/tokens`)
- [x] **의존성 주입(DI) 컨테이너 도입:** `World.js`에 집중된 로직을 해소하기 위해 DI 패턴을 적용, 각 시스템(Entity, Weather, Render)을 느슨하게 결합.
- [x] **플러그인 아키텍처 구축:** 새로운 직업, 동물, 건물 종류를 코어 엔진 수정 없이 외부 설정 파일이나 플러그인 형태로 무한히 주입(Inject)할 수 있는 모듈화 시스템 설계 (`PluginManager` 도입).
- [x] **SharedArrayBuffer (SAB) 전면 도입:** 인구 수 대규모 증가(10,000명 이상) 시 Worker와의 통신 오버헤드 제거를 위해 개체 상태(위치, 체력 등)를 공유 메모리에서 직접 Read/Write 처리. (이미 구현됨)
- [ ] **청크(Chunk) 공간 분할 및 동적 로딩:** QuadTree를 넘어 월드를 청크 단위로 나누고, 뷰포트(Camera)에 보이지 않는 청크의 렌더링 및 불필요한 AI 연산을 스킵하는 컬링(Culling) 시스템 고도화.
- [x] **청크(Chunk) 공간 분할 및 동적 로딩:** QuadTree를 2D 그리드 배열 청크 단위로 대체, 가비지 컬렉션(GC) 부하를 원천 차단하여 10,000개 이상의 개체 렌더링/컬링 최적화.

### Phase 2: 디자인 및 그래픽 렌더링 혁신 (Design & Graphics)
- [x] **스프라이트(Sprite) 렌더링 시스템:** 단순 캔버스 도형(arc, rect) 렌더링을 탈피하고, 개체별/상태별 픽셀 아트 스프라이트 시트 애니메이션 지원 (Renders 모듈 확장 및 `SpriteManager` 도입).
- [x] **다이나믹 파티클(Particle) 엔진:** 각 직업(건축가 망치질, 광부 곡괭이 등) 상호작용 시 파티클 방출 및 마법/재해 이펙트 처리를 위한 범용 `ParticleSystem` 구축 및 코어 연동 완료.
- [x] **조명 및 셰이더(Shader) 효과:** TimeSystem의 낮/밤 변화에 따른 동적 그림자(Dynamic Shadow) 생성 및 계절/날씨에 따른 색조 보정(Color Grading) 필터를 처리하는 `LightingSystem` 구축 완료.

### Phase 3: 개체 상호작용 및 생태계/사회 확장 (Interaction, Job, Village, Building)
- [x] **지능형 상호작용(AI Interaction):** 개체 간 조우 시 대화(말풍선 이모지)를 주고받는 `InteractionSystem` 기본 렌더링 파이프라인 및 백그라운드 Worker 연동 완료.
- [x] **지능형 상호작용 심화:** 밤이 되면 마을 주민들이 집으로 돌아가 잠을 자고 체력을 회복하는 Night Routine AI 패턴 고도화 완료.
- [x] **자원 생산 및 채집 고도화:** 우유, 고기 등 신규 자원을 추가하고 채집/인벤토리 연동을 고도화.
- [x] **목장(Ranch) 인프라 추가:** 목장 건물을 통해 젖소(Cow)를 기르고(번식), 주기적으로 우유를 채집하며, 늙은 소는 도축(Meat)하여 식량으로 전환하는 자원 사이클 구현 완료.
- [x] **건물 티어(Tier) 및 마을 인프라:** 자원 누적치(목재 150, 석재 50 이상)에 따라 건물의 티어(Tier)가 진화하는 업그레이드 시스템 구축 완료.
- [ ] **국가 간 외교 및 전쟁 시스템:** 마을이 국가로 성장한 후, 국경이 맞닿은 타 세력 간의 동맹/적대 수치(Diplomacy) 연산 및 주기적 국지전 AI 구현.
- [x] **생태계 다양성 확장:** 토끼, 늑대, 코끼리 등 10종 이상의 신규 동물과 다이아몬드, 우라늄, 루비 등 11종의 신규 광물/자원 대규모 추가 완료.
- [x] **식량 확보 시스템 버그 수정 (4종):** ①EAT DRIVE 우선순위 역전(굶으면서 자는 버그), ②Harvesting 수확물 인벤토리 미저장, ③성인 크리쳐 직업 배정 공백(촌장 10% 확률 이슈), ④Farmer 비옥도 부족 시 fallback 부재 — 4가지 복합 버그 전부 수정 완료.
- [x] **시장(Market) 시스템 구축:** `MarketSystem.js` 기반 마을 간 자원 교역 엔진, `MARKET` 건물 타입(🏪) + gold 수입 효과, `MERCHANT` 직업(상인 💰) + `TRADING` 상태 액션, `VillageSystem`의 인구 10명 이상 시 자동 건설 AI 트리거 구현 완료.
- [x] **가문(Family) 시스템 구축:** `FamilySystem.js` 기반 한국식 성씨 50종 랜덤 배정, 번식 시 부계 성씨 계승(70% 부계/30% 모계), 성씨별 선천적 특성 보너스(근면/민첩/용맹), Inspector UI에 '김씨' 형태로 표시 완료.

### Phase 4: 고급 모니터링 및 상태창 UI/UX (Monitoring & Deep Inspector)
- [x] **심층 인스펙터(Deep Inspector) 창:** 개체 클릭 시 단순 능력치뿐만 아니라 레벨 대비 최대 경험치(MaxExp), 작업 속도 효율성(Efficiency) 등의 심층 연산 스탯을 실시간 시각화하도록 버퍼 파이프라인 고도화 완료.
- [x] **실시간 글로벌 이벤트 티커(Event Ticker):** 좌측 하단에 `GameChatLog` 뷰를 구축하여, 주민들의 일상 대화(말풍선) 및 국가 건국, 건물 티어 업그레이드, 레벨업 등의 시스템 이벤트를 스트리밍하도록 구현 완료.
- [ ] **데이터 오버레이 히트맵(Data Overlay):** 신(God) 모드 UI에서 토글 버튼으로 미니맵 상에 온도 분포, 토지 비옥도, 세력권 경계, 인구 밀집도 등을 직관적으로 볼 수 있는 히트맵 렌더링 뷰 추가.
- [ ] **상세 통계 대시보드(Dashboard):** 10년, 100년 주기의 인구 증감 추이, 자원 소모량 등을 보여주는 `Chart.js` (또는 유사 라이브러리) 기반의 그래프 모달 패널 개발.
